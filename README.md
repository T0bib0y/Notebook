# 📓 Notebook

**The cleanest, ad-free text editor & notebook — now with PWA + Search.**

A beautiful, privacy-first Markdown editor that runs 100% in your browser. No ads. No tracking. No accounts. Your notes live only on your device.

## ✨ Features

- **Powerful full-text search** across titles and content
- **Progressive Web App (PWA)** — installable & works offline
- Multiple notes with clean sidebar
- Live Markdown preview (split view)
- Instant auto-save to localStorage
- Dark & Light themes
- Export to `.md` or `.txt`
- Word & character count
- Keyboard shortcuts

## 🚀 Quick Start

### Run locally (recommended)
1. Open `index.html` in any modern browser
2. Start writing — notes save automatically

### As PWA (best experience)
- Open `index.html`
- Click the **install icon** in the address bar (or browser menu)
- It will install like a native app
- Works offline after the first load

## 🛠️ How to Use

### Search
- Type in the search bar at the top of the sidebar
- Matches both **title** and **content**
- Results are highlighted
- Press `Esc` to clear search
- Press `/` (when not typing) to focus search

### Notes
1. Click **New** or press `Ctrl/Cmd + K`
2. Click any note to open it
3. Edit title in the top bar
4. Write Markdown — live preview on the right
5. Export using the dropdown

## 🔒 Privacy & PWA

- 100% local storage
- Service Worker enables offline use
- No data ever leaves your device
- Clean, installable app experience

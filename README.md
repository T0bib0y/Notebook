# 📓 Notebook

**The cleanest, ad-free text editor & notebook.**

A beautiful, privacy-first Markdown editor that runs 100% in your browser. No ads. No tracking. No accounts. Your notes live only on your device.

## ✨ Features

- **Multiple notes** with a clean sidebar
- **Live Markdown preview** (split view on desktop)
- **Instant auto-save** to localStorage
- **Dark & Light themes**
- **Export** to `.md` or `.txt`
- **Word & character count**
- **Keyboard shortcuts** (Ctrl/Cmd + K = new note)
- Fully responsive
- Built with Tailwind + Marked.js

## 🚀 Quick Start

### Option 1: Use it instantly (Recommended)
1. Open the `index.html` file in any modern browser
2. Start writing — everything saves automatically

### Option 2: GitHub Pages
Visit the live version (if Pages is enabled) or simply open `index.html` locally.

## 🛠️ How to Use

1. Click **New** in the sidebar to create a note
2. Click any note in the list to switch
3. Edit the title by clicking on it in the top bar
4. Write Markdown in the left pane — see live preview on the right (desktop)
5. Use the **Export** button to download your note
6. Toggle theme with the moon/sun icon

**Keyboard Shortcuts:**
- `Ctrl/Cmd + K` → Create new note
- `Ctrl/Cmd + S` → Force save

## 🔒 Privacy First

This is a truly ad-free experience:
- Zero external requests after initial CDN load for libraries
- All data stays in your browser's `localStorage`
- No analytics, no cookies, no telemetry
- Open source and self-hostable

# 📓 Notebook

**Clean • Organized • Mobile-Friendly Private Markdown Notebook**

A beautiful, fully-featured, ad-free text editor with folders, tags, drag & drop, duplication, and excellent mobile support.

## ✨ Features

- Custom folders + Drag & Drop
- Note duplication
- Tags system
- Great mobile responsiveness (Editor/Preview toggle on mobile)
- Import / Export JSON backups
- PWA (installable + offline)
- Live Markdown preview

## 📱 Mobile Experience
- Sidebar works well on phones
- Tap the eye icon to toggle Preview on mobile
- Fully responsive layout

## 🛠️ Useful Actions
- **Duplicate Note**: Available in the Export menu
- **Drag notes** onto folders to reorganize
- **Create custom folders** with the + New button

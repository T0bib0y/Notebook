# Notebook

**Ad-free, privacy-first text editor and notebook.**

A clean, simple, and fully offline Markdown editor that runs entirely in your browser. No ads, no tracking, no accounts — your notes stay on your device using localStorage.

## Features

- 📝 Multiple notes with sidebar
- ✍️ Live Markdown preview (split view)
- 💾 Auto-save to local browser storage
- 🌙 Dark / Light theme toggle
- 📤 Export notes as Markdown (.md) or Plain Text (.txt)
- 🔢 Word & character count
- ⌨️ Keyboard shortcuts
- 📱 Responsive design, works great on desktop & mobile
- 🔒 100% private — nothing leaves your browser

## Getting Started

1. Open `index.html` in your browser (or visit the GitHub Pages URL once enabled)
2. Start typing — your notes are saved automatically
3. Create new notes from the sidebar
4. Toggle between Editor and Preview

## How to Use

- Click **+ New Note** to create a new note
- Select a note from the list to edit it
- Use the preview pane to see rendered Markdown
- Export your current note using the buttons in the header

## Privacy

This editor is completely ad-free and does not send any data anywhere. All notes are stored locally in your browser's localStorage.
